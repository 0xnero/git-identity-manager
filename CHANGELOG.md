# git-idm v0.2

New features:

- Added short commands `git idm rm` and `git idm ls` for `remove` and `list`

Bugfixes:

- Avoid removing an empty identity.
- Document `git idm version`.

# git-idm v0.1 (initial release)

Released with commands:

```
git idm active
git idm add
git idm list
git idm remove
git idm uninstall
git idm use
git idm version
```
